Digitizing Intimacy

A Computational and Network-Based Model of the T.S. Eliot–Emily Hale Correspondence (1914–1957)

About Methods Topics Network Letters Data

Project Overview

This funded-style digital humanities platform transforms Eliot's 1,133 letters into a structured computational archive. Through sentiment modeling, topic modeling, and relational graph analysis, the project reconstructs the emotional and intellectual architecture of modernism.

Methodological Framework

Corpus Processing

OCR cleaning, tokenization, metadata tagging, and normalization.

Sentiment Modeling

Lexicon-based scoring aligned with historical events.

Topic Modeling

LDA visualization of recurring themes across decades.

Graph Theory

Entity extraction and centrality measurement via network modeling.

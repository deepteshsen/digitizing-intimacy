Project Overview

This project reimagines the 1,133 letters written by T.S. Eliot to Emily Hale as a computationally structured archive. Inspired by the Republic of Letters, it integrates corpus linguistics, network analysis, and temporal visualization to examine structural patterns of intimacy, religious transformation, and intellectual affiliation across four decades.

Digital Architecture

The full project will be hosted on Omeka as an archival interface integrating:

  • Voyant corpus visualizations (embedded)
  • Gephi network exports (interactive Sigma.js)
  • TimelineJS chronological navigation
  • Searchable metadata (date, theme, sentiment score)
